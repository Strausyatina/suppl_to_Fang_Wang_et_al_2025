R scripts required to produce figures related to differential gene expression analysis
- MA_plot_Volcano_plot.R: reproduces MA and Volcano plots for all pairwise comparisons using edgeR result files as input
- PCA_plot.R: reproduces PCA plots for all comparisons using CPM count files as input
- VennDiagramms.R: reproduces VennDiagramms for all comparisons using edgeR result files as input
R sessionInfo files:
- MA_Volcano_sessionInfo.txt: sessionInfo file related to MA_plot_Volcano_plot.R
- PCA_sessionInfo.txt: sessionInfo file related to PCA_plot.R
- VennDiagram_sessionInfo.txt: sessionInfo file related to VennDiagramms.R
